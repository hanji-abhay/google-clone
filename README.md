# Google Homepage Clone

This project is a simple clone of the Google homepage built using HTML, CSS, and JavaScript. 
It was created as a practice project to strengthen frontend development skills and understanding of web design structure.

## Features
- Replicates Google’s homepage layout
- Responsive design (optional: if you added any)
- Interactive elements using JavaScript (e.g., buttons or search bar simulation)
- Organized assets folder for images and icons

## Project Structure
google-clone/
│
├── index.html
├── style.css
├── script.js
└── assets/ (images/icons used in the project)

## Skills Practiced
- HTML semantic structure
- CSS styling and layout
- JavaScript interactivity
- Project organization and folder structure

## How to Run
1. Clone or download the repository.
2. Open `google-clone.html` in your browser.
3. Explore the Google homepage clone!
